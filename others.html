About Jordan Curve Theorem   Every Jordan curve, a non-self-intersecting continuous loop in the plane, separates the plane into exactly two components. This is a theorem that is extremely easy to understand yet it hard to prove. Just forget about the proof which involves the connected components in topology, enjoy some artworks using the Jordan curve in this paper.

  • F. Ross and W. T. Ross, "The Jordan curve theorem is non-trivial," J. Math. Arts. 5, 213 (2011).
